Inventory Tracking Application

Description:
This project reads a lists and gets the count based on a searched word or creates a list or histogram with the item count

The program does the requirements as outlined in the 210 Programming languages course. This is implementing a class and reading an external file and creating an external backup. This also utilizes maps to store an object with a keyword.

One idea is to tie this to a database to read through a chaging set of values.

The most difficult part was truly understanding how MAPS work. I had used them before to use keywords to deliver functions. I didn't realize they work like dictionaries and just hold a phrase to run a return the key. They are powerful to automate or make factories. Using resources like explanation videos on youtube was definitely helpful.

In general reading and writing to files is very useful across many different structures.

Documentation and seperating out the class and header files has been done as throughroughly as possible.
